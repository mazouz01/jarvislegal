# Jarvis Legal
Test technique backend Symfony - API platform

1- Make sure you have Composer installed and run: composer install <br>
2- Edit .env file with your own database infos and run:<br><br>

php bin/console doctrine:database:create <br>
php bin/console doctrine:migrations:migrate <br>
<br>
**Start the built-in web server** <br>
3- download symfony installer from here: https://symfony.com/download <br>
4- run command: symfony serve -d  <br>
<br>
5- go to:  http://localhost:8000/api <br>
